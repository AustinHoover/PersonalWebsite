Signal Server Project
The goal of this project is to create a server which can facilitate secure communication between various other scripts/programs regardless of NAT/other network issues.
Approximate Start Date: August 2019
Source code: Github
Currently "Completed":
  • Encrypted Sockets
  • Division of Communication channels
  • Inter-"process" communication
Future goals:
  • User Authentication
  • Client software
  • Client library
  • Server scheduling optimization
